# segment_ios

## Set Up
1. Download Cocoapods here: https://cocoapods.org/
```
$ sudo gem install cocoapods
```

2. Create a Podfile in your project folder (skip this if Pods folder exists)
```
pod init
```

3. Install the pods
```
pod install
pod update
````

Reference: https://codewithchris.com/cocoapods/
